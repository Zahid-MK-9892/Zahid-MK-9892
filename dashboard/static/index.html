WhatsApp Setup Guide
Uses Twilio (free tier available)
  1. 1Go to twilio.com → sign up free
  2. 2Go to Messaging → Try it out → Send a WhatsApp message
  3. 3Send the join code from your WhatsApp to the Twilio sandbox number
  4. 4Copy your Account SID and Auth Token from Twilio Console
  5. 5Add them to your .env file and restart the dashboard
